Deploy

Запуск за 60 секунд

Пять шагов. Без CLI. Без оплаты. Без сложных настроек.

1

Аккаунт GitHub

Зарегистрируйтесь на github.com (бесплатно)

2

Откройте шаблон

Перейдите на feekool.github.io — готовый репозиторий для испанского

3

Fork репозитория

Нажмите "Use this template" → "Create a new repository". Назовите: mi-espanol или spanish-forum

4

Включите GitHub Pages

Settings → Pages → Source: main → Save. Через минуту сайт онлайн.

https://ваш-ник.github.io/mi-espanol
5

🇪🇸 Настройте контент

Редактируйте файлы в репозитории: добавьте «¡Bienvenidos!», разделы Gramática/Vocabulario, уровни A1-C1. Готово.

📦 В шаблоне уже есть: готовая структура разделов, примеры тем, конфигурация навигации. Просто раскомментируйте нужные блоки и заполните своими материалами.
