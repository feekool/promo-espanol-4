Запустите форум для испанского за 60 секунд. Без серверов.  

Описание
--------
Статический, быстрый и бесплатный форум, готовый к развёртыванию на GitHub Pages, который позволяет сразу начать практиковать испанский в структурированной среде с первого дня. Этот шаблон оптимизирован для преподавателей и студентов: он сочетает в себе форумные треды для живой практики, централизованный контент-хаб и полностью статическую архитектуру без бэкенда.  

Ключевые призывы к действию
---------------------------
- 🌊 Открыть демо  
- 📦 Fork шаблон  
- ⚡ Deploy за 1 минута на GitHub Pages  
- 🆓 Open Source — бесплатно навсегда  

Ключевые возможности
--------------------
- Архитектура для эффективного обучения: создан для преподавателей и студентов, которые ценят скорость, структуру и контроль.  
- Живая практика: треды для обсуждения, коррекции ошибок и ролевых диалогов; студенты пишут — преподаватель комментирует.  
- Контент-хаб: уроки, грамматика, аудио и тесты, структурированные по CEFR (A1→C1). Markdown-файлы легко редактировать без программирования.  
- Zero Backend: без серверов, баз данных или платных подписок — работает на GitHub Pages, бесплатно и безопасно.  
- Полный контроль: настраивайте темы, разделы, приветствия и добавляйте блоки культуры, челленджи и глоссарий.

Детали возможностей
-------------------
- Живая практика — треды позволяют организовать исправления прямо в обсуждении, проводить ролевые диалоги и давать обратную связь от преподавателей; все сообщения индексируются и сохраняются.  
- Контент-хаб — структура по уровням CEFR делает курс последовательным: уроки, грамматические заметки, аудиофайлы и тесты доступны в виде Markdown-файлов, которые легко редактировать через GitHub.  
- Zero Backend — никакой настройки серверов: сайт развертывается на GitHub Pages, автоматически масштабируется и не требует оплаты за хостинг или БД.  
- Кастомизация — темы, разделы и приветственные сообщения настраиваются; можно добавить культурные блоки, челленджи и персональные разделы для дневников.  

Трек обучения по CEFR (уровни)
------------------------------
- A1 — Основы и фразы.  
- A2 — Диалоги и бытовые ситуации.  
- B1 — Контекстное понимание и грамматика.  
- B2 — Аргументация и свободное выражение мыслей.  
- C1 — Свободное владение и гибкость языка.  

Pro tip: создайте раздел «Mi diario» для записи новых слов и отслеживания прогресса; трек мотивации работает лучше любых приложений.  

Deploy: запуск за 60 секунд
--------------------------
Пять простых шагов, без CLI и без оплаты:  
1) Зарегистрируйтесь на GitHub (если ещё не сделали).  
2) Откройте шаблон: перейдите на репозиторий feekool.github.io (готовый репозиторий для испанского).  
3) Fork репозитория: нажмите "Use this template" → "Create a new repository". Назовите репозиторий, например, mi-espanol или spanish-forum.  
4) Включите GitHub Pages: Settings → Pages → Source: main → Save. Через минуту сайт будет онлайн по адресу https://ваш-ник.github.io/mi-espanol.  
5) Настройте контент: редактируйте файлы в репозитории — добавьте «¡Bienvenidos!», разделы Gramática/Vocabulario и уровни A1–C1.  

Что уже включено в шаблон
-------------------------
Готовая структура разделов, примеры тем и конфигурация навигации; вам остаётся раскомментировать нужные блоки и заполнить их своими материалами.
